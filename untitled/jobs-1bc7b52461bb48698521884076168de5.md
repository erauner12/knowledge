## jobs

## Overview

* jobs mean pods which will terminate at some point
  * run-to-complete
* deleting jobs after completion frees up resources
* Mostly used for parallell processing
  * batch jobs

```yaml
apiVersion: batch/v1
kind: Job
metadata:
  name: job-pi
spec:
  template:
    spec:
      containers:
      - name: job-container
        image: perl
        command: ["perl",  "-Mbignum=bpi", "-wle", "print bpi(2000)"]
      restartPolicy: Never
  backoffLimit: 4
```
